# 图片资源说明

此目录用于存放网站所需的图片资源。

## 目录结构

```
images/
├── banner/           # 轮播图和视频封面
├── qrcodes/          # 二维码图片
├── icons/            # 图标文件
└── examples/         # 算例预览图
```

## 图片要求

### Banner轮播图
- 尺寸：1920x1080 像素（推荐）
- 格式：JPG 或 PNG
- 文件大小：建议不超过 500KB
- 命名：`banner-1.jpg`, `banner-2.jpg` 等

### 视频封面
- 尺寸：1920x1080 像素（16:9比例）
- 格式：JPG
- 命名：`video-poster.jpg`

### 二维码
- 尺寸：400x400 像素
- 格式：PNG（支持透明背景）
- 命名：
  - `wechat-qr.png` - 微信公众号二维码
  - `qq-qr.png` - QQ群二维码

### 算例预览图
- 尺寸：600x400 像素（3:2比例）
- 格式：JPG
- 文件大小：建议不超过 100KB
- 命名与对应算例一致

## 图片优化建议

1. 使用 TinyPNG 或 ImageOptim 等工具压缩图片
2. 轮播图可使用 WebP 格式以获得更好压缩率
3. 为不同分辨率提供响应式图片

## 占位符说明

当前使用 Font Awesome 图标作为占位符，如需替换为实际图片：
1. 将图片文件放入对应目录
2. 更新 HTML/JS 中的图片引用路径
3. 确保图片路径正确无误
