# 视频资源说明

此目录用于存放FPMCAE软件介绍视频。

## 视频要求

### 主视频
- 文件名：`fpmcae-intro.mp4`
- 格式：MP4 (H.264编码，兼容性最佳)
- 分辨率：1920x1080 (1080p)
- 时长：建议 3-5 分钟
- 文件大小：建议不超过 50MB

### 视频封面
- 放置在 `images/banner/video-poster.jpg`
- 尺寸：1920x1080 像素

## 视频内容建议

1. **开场** (0:00-0:30)
   - FPMCAE Logo展示
   - 软件名称和版本

2. **简介** (0:30-1:00)
   - 开发背景
   - 核心算法介绍（有限质点法）

3. **功能展示** (1:00-2:30)
   - 界面概览
   - 建模功能
   - 求解器设置
   - 结果可视化

4. **应用案例** (2:30-4:00)
   - 典型工程应用展示
   - 计算结果动画

5. **结尾** (4:00-5:00)
   - 联系方式
   - 下载指引

## 技术参数

```bash
# 推荐的 ffmpeg 转码参数
ffmpeg -i input.mp4 -c:v libx264 -preset slow -crf 23 -c:a aac -b:a 128k fpmcae-intro.mp4
```

## 浏览器兼容性

- Chrome/Edge: 完全支持
- Firefox: 完全支持
- Safari: 完全支持（建议使用 H.264）
- IE11: 有限支持（需安装插件）
